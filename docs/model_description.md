# Model Specification

Spatial Ecology RS implements a stochastic patch-dynamics model for spatial trophic systems. The model is designed for computational experiments over connectivity, disturbance, and trophic interaction parameters.

## State Space

The landscape is a two-dimensional grid of habitat patches. Patch `i` at row `r` and column `c` has state:

```text
X_i(t) = [N_i(t), P_i(t), V_i(t), R_i(t), T_i(t), D_i(t)]
```

where:

- `N_i(t)` is prey abundance.
- `P_i(t)` is predator abundance.
- `V_i(t)` is vegetation biomass.
- `R_i(t)` is rainfall multiplier.
- `T_i(t)` is local temperature.
- `D_i(t)` is disease pressure.

Each patch also has carrying capacity `K_i`. Biological abundance variables are constrained to be nonnegative after each timestep.

## Local Trophic Dynamics

Within each patch, the model applies a discrete-time update:

```text
vegetation_growth = r_v V_i max(0, 1 - V_i / K_i)
                    rainfall_factor temperature_factor

grazing = min(g_v N_i V_i / K_i, V_i + vegetation_growth)

prey_births = r_n N_i forage_factor disease_factor
prey_deaths = d_n N_i
predation = min(alpha N_i P_i, N_i + prey_births)

predator_births = beta predation
predator_deaths = d_p P_i

V_i(t + 1) = V_i(t) + vegetation_growth - grazing
N_i(t + 1) = N_i(t) + prey_births - predation - prey_deaths
P_i(t + 1) = P_i(t) + predator_births - predator_deaths
```

Parameters:

- `r_v`: vegetation growth rate.
- `g_v`: grazing coefficient.
- `r_n`: prey birth rate.
- `d_n`: prey mortality rate.
- `alpha`: predation coefficient.
- `beta`: predator conversion efficiency.
- `d_p`: predator mortality rate.
- `K_i`: carrying capacity.

This formulation is intentionally parsimonious. It is appropriate for simulation-method development and parameter sensitivity analysis, not for direct empirical forecasting without calibration.

## Spatial Coupling

Patches are connected by a von Neumann neighborhood: north, south, east, and west. During initialization, each undirected edge is removed with probability `fragmentation_rate`.

For each connected patch:

```text
migrants = abundance * migration_rate
```

Migrants are distributed across available neighbors in proportion to edge weights. In the current implementation all retained edges have equal weight. Migration is conservative: total prey and predator abundance before and after the migration step must match within numerical tolerance.

Prey dispersal increases under vegetation scarcity. Predator dispersal increases under prey scarcity.

## Stochastic Disturbance

Stochastic events are controlled by a seeded random number generator:

- Drought reduces vegetation biomass and rainfall.
- Disease reduces prey abundance and increases disease pressure.
- Temperature anomalies perturb local temperature.
- Habitat disturbance reduces vegetation biomass.

Each event has a probability and severity parameter in the TOML configuration. Event labels are recorded per patch and timestep.

## Extinction Criteria

Extinction is evaluated at the landscape scale by summing abundance over all patches:

```text
N_total(t) = sum_i N_i(t)
P_total(t) = sum_i P_i(t)
```

Prey or predators are marked extinct when total abundance falls below the configured threshold:

```toml
[thresholds]
prey_extinction_threshold = 1.0
predator_extinction_threshold = 1.0
```

Sweep configurations may use higher functional thresholds to study loss of ecological viability before complete numerical extinction.

## Numerical Validation

The validator checks:

- no negative abundance values beyond floating-point tolerance
- no NaN or infinite values
- vegetation does not exceed carrying capacity
- migration conserves prey and predator totals
- patch count remains constant

These checks are intended to catch numerical instability and implementation errors during parameter sweeps.

## Known Limitations

- Parameters are synthetic and not calibrated to a specific empirical system.
- Timestep duration is unitless.
- Carrying capacity is fixed per scenario rather than dynamically inferred.
- Connectivity is currently generated by random edge deletion rather than by an empirical landscape graph.
- Event processes are independent Bernoulli draws and do not include spatial autocorrelation.

These limitations are deliberate simplifications for an extensible computational ecology model. They also identify natural directions for future calibration and methodological work.
